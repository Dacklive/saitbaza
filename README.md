# Речная прохлада

Готовые файлы статического сайта для загрузки на GitHub Pages.

## Как загрузить

1. Создайте репозиторий на GitHub.
2. Загрузите в корень репозитория все файлы из этой папки:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.nojekyll`
   - папку `assets`
3. Откройте настройки репозитория: `Settings` -> `Pages`.
4. В разделе `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Нажмите `Save`.

Через 1-3 минуты сайт будет доступен по ссылке GitHub Pages.
